Hello web audio

A minimal command-based live coding engine. Enter a space-separated list of duration@pitch tokens — both sides are JavaScript expressions.

Example: 1+1@220 2@220*2 1*3@270
Repeat: 2[1@220 2@330] plays 1@220 2@330 twice.
Use Track 1 and Track 2 to layer two patterns simultaneously.
